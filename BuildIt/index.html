Inspiration for my Website Styling

Updated Figma Screenshot

Fonts

My header font is called Contrail One.

My label font is called Open Sans Condensed.

My paragraph text is called Barlow.

I will be using a color pallete of 5 colors.

  • Color 1: rgb(242,121,143)
  • Color 2: rgb(242,92,147)
  • Color 3: rgb(36,28,64)
  • Color 4: rgb(242,232,220)
  • Color 5: rgb(242,187,187)

My links

The font for my links will be the same as my labels: Barlow

  • The original state will be black
  • The hover state will be color 1
  • The clicked state will be bolded black.

They will all be underlined!

My Buttons

My site will have a button with three different states of style.

  • The first button is color 3 with the text color as color 5.
  • The second state being color 2 and the text color is color 4.
  • The final state is color 4 and the text color as color 2.

Within the button, all text will be consistant and its font called Open Sans Condensed.

My Text Input

My text input will use 3 colors from my pallete, one as the background, another as the inner boarder, and lastly for the text and icon. The text font is called Open Sans Condensed

  • The background is color 5.
  • The inner border is color 3.
  • The text and icon is color 2.
